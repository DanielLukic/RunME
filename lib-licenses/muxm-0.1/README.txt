Introduction
------------

The goal of the MuSound project is to provide better audio capabilities to 
J2ME/MIDP2 applications. 

MuXM is a player library for J2ME/J2SE which allows to play MOD and XM files.

Licencing

MuXM is released under LGPL license. See LICENSE.txt file.

Distribution structure:
-----------------------
	
	|-- demo: contains the MuXM demo for MIDP2 
	|-- dist: contains the MuXM library
	|-- docs: contains the documentation in html format
	|-- src: contains the sources
